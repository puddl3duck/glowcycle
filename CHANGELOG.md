# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- AI-powered skin analysis feature
- Advanced analytics dashboard

- Mobile responsive improvements

## [2.0.0] - 2026-02-14

### Added
- Complete dark mode implementation with 100% coverage
- Time-based personalization (morning, afternoon, night)
- Auto theme switching based on time of day
- Comprehensive documentation (7 documents)
- CI/CD pipeline with GitHub Actions
- Code of Conduct
- EditorConfig for consistent coding style
- Professional project structure

### Changed
- Reorganized frontend into dedicated folder
- Moved documentation to `docs/` folder
- Moved scripts to `scripts/` folder
- Updated README with comprehensive information
- Improved .gitignore with extensive rules

### Fixed
- Back button arrows now white for better visibility
- Cycle wheel labels now white in dark mode
- Skin metrics values now solid pink (no transparency)
- Scanner instructions fully visible in dark mode
- Consent popup fully visible with solid background
- All icons with natural brightness (0.95x)

## [1.0.0] - 2026-02-13

### Added
- Cycle tracking with visual wheel
- Journal & mood tracking
- Skin tracking (manual entry)
- Time-based greetings and prompts
- Theme toggle functionality
- localStorage persistence
- Responsive design

### Features
- **Cycle Tracking**
  - Visual cycle wheel with phase indicators
  - Phase-specific tips and recommendations
  - Predictions for next period and ovulation
  
- **Journal & Mood**
  - Daily mood tracking with emoji interface
  - Energy level monitoring
  - Thought journaling with word count
  - Customizable tags
  
- **Skin Tracking**
  - Manual skin condition logging
  - Personalized skincare routine recommendations
  - Time-based routines (AM/PM)

### Technical
- Python backend with AWS Lambda
- DynamoDB for data storage
- S3 for static assets
- AWS CDK for infrastructure
- Vanilla JavaScript frontend

## [0.1.0] - 2026-02-01

### Added
- Initial project setup
- Basic HTML structure
- CSS styling foundation
- JavaScript functionality skeleton
- Backend handlers structure
- Infrastructure setup with CDK
